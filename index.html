I can't give rewards for their efforts quite yet, but I empower them to be worthy of rewards. Paying forward what I learned from my seniors and managers but adding a lot of my own touches.

I find it really rewarding when I get to solve it, more so if I made it as efficient as I can. Then going beyond my own capability, and being open for more optimizations suggestions.

I also get lost hours, days (even weeks) on end web developing, but more so in web designing. As an aesthetic, seeing my work get better and better with each code I add is like a never ending dopamine hit. It'll hook me to it until I'm finally, I mean REALLY finally, happy with the result. Just before the point where adding more, will make it less.
